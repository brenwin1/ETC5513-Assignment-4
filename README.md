# ETC5513-Assignment-4

In this assignment we have investigated Crime in Minneapolis. We have analysed the crime incidence, the neighborhoods where the crime incidence is high, crime in Minneapolis over time, the use of force by police and the areas where force is used by police. 

In performing this analysis, we used data sets from the City of Minneapolis. We used the force of use dataset, the shapefiles and the crime incident data. We used an MIT license.
